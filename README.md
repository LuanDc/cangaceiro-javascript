# Cangaceiro Javascript

Projeto feito baseado no livro [Cangaceiro Javascript](https://www.casadocodigo.com.br/products/livro-cangaceiro-javascript) 
com a finalidade de aprender novos patterns e algoritmos baseados em Programação Orientada a Objetos.

## Rodando o projeto localmente

**Servidor**

O diretório raiz do projeto vem com uma API para realizar requisições e buscar dados que vão ser renderizados no cliente. 

Navegue até a pasta server

```
    cd server
```

Instale as dependência via NPM.

```
    npm install
```

Agora digite o comando em seu terminal. 

```
    npm start
```

**Cliente**

Navegue até a pasta client

```
    cd client
```

Instale as dependência via NPM.

```
    npm install
```

Agora digite o comando em seu terminal. 

```
    npm start
```

**Acessando pelo browser**

![Captura de tela de 2020-02-11 13-47-55](https://user-images.githubusercontent.com/42556949/74258515-363e0280-4cd5-11ea-8d2f-1eade9805099.png)

Quando você rodar o seu servidor de desenvolvimento da pasta client uma mensagem no console irá mostrar qual porta está rodando sua aplicação.
Digite ```http://localhost:8080``` (Se a sua porta 8080 estiver como padrão) no seu browser e perfeito, tudo funcionando.
